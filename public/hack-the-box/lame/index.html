Résolution de la box Lame

Introduction

La box Lame de Hack The Box est une box Linux de difficulté “Easy”. Cette box fait aussi parti du parcours “Beginner Track” du mode “Tracks”.

Enumeration

Pour commencer on énumère les ports ouverts pour trouver les services distants.

FTP Anonyme

On voit que le service FTP est accessible en anonyme. On tente une connexion pour vérifier s’il y a des fichiers qui pourraient nous intéresser.

Il n’y a rien dans le répertoire. Nous vérifions si nous avons les droits d’écriture

Ici, impossible d’ajouter du contenu, alors on continue l’énumération

Enumération des versions

Avec searchsploit on commence par énumérer les vulnérabilités connues en fonction des versions des différents services. Ici deux services m’ont interessé : FTP et SMB.

On voit sur l’image ci-dessous qu’il existe un exploit sur cette version de FTP. De plus il est diponnible sous metasploit.

Je regarde aussi s’il existe des exploits disponnible sous SMB.

Ici on peut voir qu’il existe aussi une vulnérabilité de type RCE dans la version de samba et que l’exploit est présent dans Metasploit.

Exploit

Je me suis d’abord interessé à la vulnérabilité sur FTP car c’est la première que j’ai trouvé.

FTP via Metasploit

J’ai d’abord lancé metasploit avec la commande msfconsole

On recherche d’abord le chemin de l’exploit au sein de metasploit.

On configure ensuite notre attaque.

Et on la lance.

Ici on voit qu’aucune session n’a été créé.

FTP via script

J’ai voulu savoir si l’erreur venait de metasploit ou si la machine bloquait cette attaque. Pour cela j’ai téléchargé un script sur le git suivant : https://github.com/ahervias77/vsftpd-2.3.4-exploit

Ici le script n’a pas fonctionné non plus, je l’ai donc annulé pour me concentrer sur la vulnérabilité SMB.

Samba via Metasploit

Comme pour FTP on sélectionne l’exploit correspondant à la vulnérabilité.

Ensuite on configure l’attaque en indiquant notre interface réseau dans LHOST et la machine distance dans RHOSTS.

Enfin on lance l’attaque et on énumère les dossiers pour trouver les deux flags.
