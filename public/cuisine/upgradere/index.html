Upgrade Reverse Shell avec ZSH

On utilise python

python3 -c 'import pty;pty.spawn("/bin/bash");'

D’abord on met notre shell en arrière-plan

CTRL + Z

on note le nombre de ligne et colonne

stty -a

Dans le shell de notre machine on écrit

stty raw -echo; fg

Et on appuie deux fois sur entrée. Ensuite on met notre shell à la bonne taille

stty rows ROW columns COL

Puis on fait afficher les couleurs

export TERM="xterm-256color"
