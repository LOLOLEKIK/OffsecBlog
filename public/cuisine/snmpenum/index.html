SNMP - Enumeration

in CUISINE
smtp enumeration onesixtyone snmpwalk
1 min read

Introduction

SNMP est un protocole permettant de superviser un réseau via les ports 161 et 162 (pour les traps).

Onesixtyone

Enumération d’utilisateur

onesixtyone -c <fichier avec les communautés à tester> -i <fichier avec les ips>

onesixtyone <ip> <communauté>

SNMwalk

Enumération de l’arbre MIB entièrement

snmpwalk -c <communauté> -v <version> <ip>

Enumération des utilisateurs windows

Il existe plein de branches à l’arbre MIB à interroger. On peut demander les ports ouverts, les services présents, les utilisateurs etc…

snmpwalk -c <communauté> -v <version> <ip> 1.3.6.1.4.1.77.1.2.25
