Connexion anonyme

ENUM4LINUX

Enumération par défaut

┌──(lololekik㉿Ashitaka)-[~]
└─$ enum4linux <ip>

Enumération avec toutes les options

┌──(lololekik㉿Ashitaka)-[~]
└─$ enum4linux -a <ip>

Enumération avec authentification

┌──(lololekik㉿Ashitaka)-[~]
└─$ enum4linux -u <user> -p <password> <ip>

Enumerer les partages

┌──(lololekik㉿Ashitaka)-[~]
└─$ enum4linux -S <ip>

Crackmapexec

Avec crackmapexec on peut bruteforce des accès smb pour tenter de trouver des mots de passe.

crackmapexec smb <ip> -d <domaine> -u <usersfile> -p <passwordfile>

SMBmap

Lister les répertoires

smbmap -u <user> -p <password> -d <domain> -H <ip>

Executer des commandes

smbmap -u <user> -p <password> -d <domain> -H <ip> -x <command>

Lire un répertoire

smbmap -u <user> -p <password> -d <domain> -H <ip> -r '<répertorie>'

Récupérer tous les fichiers avec une extension particulière

smbmap -u <user> -p <password> -d <domain> -H <ip> -A '(<ext1>|<ext2>|<ext3>)' -R
